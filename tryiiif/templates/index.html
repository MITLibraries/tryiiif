About TryIIIF

IIIF (the International Image Interoperability Framework) unlocks image content by providing a standard API for providers to expose images to visual tools for exploration, manipulation, annotation, combination, and more. We created this site to allow you to get a taste of what these tools (and IIIF) can do for you: enter any URL to a web-accessible image, and it will be served from an IIIF-compliant image server to either Mirador or Universal Viewer. We hope this will inspire you to consider adopting IIIF, or at least learn more about its capabilities.
